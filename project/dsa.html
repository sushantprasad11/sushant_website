ARRAYS

An array is a data structure that stores a fixed-size sequence of elements of the same data type. Each element in an array is identified by an index or a position, and the elements are stored in contiguous memory locations.

Types Of Array

1.One-Dimensional Array:

A simple array with a single row of elements.
Example: arr = [1, 2, 3, 4]

2.Multi-Dimensional Array:

Arrays with more than one dimension, such as matrices (2D arrays).
Example: arr = [[1, 2], [3, 4], [5, 6]] (2D array)

Array Representation

1. A 1D array is a linear collection of elements, where each element can be accessed by a single index.

Address of arr[0]: The first element is stored at the base address, say base_address.
Address of arr[1]: The second element is stored at base_address + size_of_element.
Address of arr[2]: The third element is stored at base_address + 2 * size_of_element.

2. A 2D array is essentially an array of arrays. It represents a matrix or a table-like structure. A 2D array can be thought of as having rows and columns.

Insertion And Deletion

In C, insertion and deletion in an array involve adding or removing elements at specific positions, and they require shifting the other elements accordingly to maintain the correct order of the array. These operations are performed in 1D arrays and can be done using loops.

Insertion:

Insertion in an array involves adding a new element at a specified index. After inserting an element, all the elements after that index must be shifted to the right to make space for the new element.

Step For Insertion

Inserting an element at a specific position:
Step 1: Shift elements to the right, starting from the last element to the position where you want to insert the new element.
Step 2: Insert the new element at the desired position.

Searching In Array In C

Searching in an array in C can be done in several ways depending on the type of search algorithm you want to use. The most common search methods are linear search and binary search.

1. Linear Search

Linear search is the simplest search algorithm. It checks each element of the array one by one, from the beginning to the end, until it finds the element or exhausts the entire array.

Steps For Searching

Step 1: Start from the first element of the array.
Step 2: Compare each element with the target element.
Step 3: If the element is found, return its index. If not, continue to the next element.
Step 4: If the search reaches the end of the array without finding the element, return a failure value (e.g., -1).

Program


output:

Element 20 found at index 3

2.Binary Search

Binary search is a more efficient search algorithm but it requires the array to be sorted. It works by repeatedly dividing the search interval in half.

Steps For Bineary Search

Step 1: Start with the middle element of the array.
Step 2: If the target element is equal to the middle element, return the index.
Step 3: If the target element is less than the middle element, repeat the search in the left half of the array.
Step 4: If the target element is greater than the middle element, repeat the search in the right half of the array.
Step 5: If the search interval becomes empty, return a failure value (e.g., -1).

Linked-List

A linked list is a linear data structure in which elements, called nodes, are connected using pointers. Unlike arrays, linked lists do not have a fixed size, and each node points to the next node in the sequence, allowing dynamic memory allocation.

Components Of Linked-List

1.Node:

A node typically contains two parts:
1.Data: The value or information that the node holds.
2.Next: A pointer that points to the next node in the list.

2.Head:

The head is a pointer to the first node of the linked list. If the list is empty, the head is NULL.

Types Of Linked-List

1.Singly Linked List:

Each node points to the next node, and the last node points to NULL.


2.Doubly Linked List:

Each node points to both the next node and the previous node.


3.Circular Linked List:

The last node points to the head (first node), making the list circular.
