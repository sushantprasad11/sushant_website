C Programming Detailed Notes

1. Introduction to C

C is a procedural programming language developed by Dennis Ritchie at Bell Labs in 1972. It is known for its efficiency, flexibility, and ability to perform low-level operations.

Features of C:

  • Portability: Programs written in C can be easily moved from one system to another with minimal modification.
  • Efficiency: C provides low-level access to memory and is highly efficient.
  • Structured: C encourages the use of functions, making code modular and easier to understand.

Uses of C:

  • System Software Development (e.g., Operating Systems like UNIX).
  • Embedded Systems (in hardware).
  • Compilers and Interpreters.

2. Overview of C

C language is a high-level language that is compiled, meaning C code is transformed into machine language by a compiler before execution. C is often referred to as a middle-level language because it combines both high-level constructs and low-level operations (such as pointers).

Key components:

  • Basic syntax: C uses simple syntax to define variables, functions, loops, and conditional statements.
  • Functions: C supports functions for breaking down complex tasks into smaller sub-tasks. The main() function is the entry point of every C program.
  • Pointers: C allows direct access to memory, offering the ability to manipulate data through addresses.
  • Control structures: C includes the usual flow-control mechanisms like if, else, switch, loops, etc.

3. Constants in C

A constant is a value that cannot change during the execution of a program.

Types of Constants:

  • Literal Constants: Fixed values (e.g., 10, 'A', 3.14).
  • Symbolic Constants: Defined using #define or the const keyword.

Example of Symbolic Constants:

#include <stdio.h>
#define PI 3.14  // Preprocessor constant

int main() {
    const int MAX = 100;  // Constant variable
    printf("PI: %.2f\n", PI);
    printf("Max value: %d\n", MAX);
    return 0;
}

4. Variables in C

A variable is a storage location that holds a value which can be modified during the execution of the program.

Types of Variables:

  • Local Variables: Declared inside a function and accessible only within that function.
  • Global Variables: Declared outside of all functions and accessible throughout the program.

Example: Variables in C

#include <stdio.h>

int main() {
    int x = 5;        // integer variable
    float y = 3.14;   // floating-point variable
    char z = 'A';     // character variable

    printf("Integer: %d, Float: %.2f, Character: %c\n", x, y, z);
    return 0;
}

5. Data Types in C

Data types define the type of data a variable can store. Common data types include:

Common Data Types:

  • int: Integer data type for storing whole numbers.
  • float: Floating-point data type for storing decimals.
  • double: For storing large floating-point numbers.
  • char: For storing single characters.

Example: Data Types in C

#include <stdio.h>

int main() {
    int num = 10;
    float pi = 3.14;
    double largeNum = 123456.789;
    char grade = 'A';

    printf("Integer: %d\n", num);
    printf("Float: %.2f\n", pi);
    printf("Double: %.3f\n", largeNum);
    printf("Character: %c\n", grade);

    return 0;
}

6. Input and Output Operations

Standard Input and Output operations are handled using the functions:

  • scanf(): Used to read input from the user.
  • printf(): Used to display output to the console.

Example: Input and Output

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);  // Reading integer input
    printf("Your age is: %d\n", age);  // Printing the value
    return 0;
}

7. Operators and Expressions

Operators are symbols that perform operations on variables and values.

Types of Operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, <, >, <=, >=
  • Logical Operators: &&&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=

Example: Using Operators

#include <stdio.h>

int main() {
    int a = 10, b = 5;
    printf("a + b = %d\n", a + b);
    printf("a - b = %d\n", a - b);
    printf("a * b = %d\n", a * b);
    printf("a / b = %d\n", a / b);
    printf("a %% b = %d\n", a % b);  // Modulus operation
    return 0;
}

Control statements are used to control the flow of execution in a program.

Types of Control Statements:

    7. Program to Generate Fibonacci Series

    This program generates the Fibonacci series up to a given number.

    #include <stdio.h>

int main() {
    int n, first = 0, second = 1, next;
    printf("Enter the number of terms: ");
    scanf("%d", &n);
    printf("Fibonacci Series: ");
    for (int i = 1; i <= n; ++i) {
        if (i == 1) {
            printf("%d, ", first);
            continue;
        }
        if (i == 2) {
            printf("%d, ", second);
            continue;
        }
        next = first + second;
        first = second;
        second = next;
        printf("%d, ", next);
    }
    printf("\n");
    return 0;
}

    10. Program to Sort an Array

    This program sorts an array of integers in ascending order.

    #include <stdio.h>

int main() {
    int n, i, j, temp;
    printf("Enter the number of elements: ");
    scanf("%d", &n);
    int arr[n];
    printf("Enter the elements: ");
    for (i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }
    for (i = 0; i < n - 1; i++) {
        for (j = i + 1; j < n; j++) {
            if (arr[i] > arr[j]) {
                temp = arr[i];
                arr[i] = arr[j];
                arr[j] = temp;
            }
        }
    }
    printf("Sorted Array: ");
    for (i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}

    17. Program to Check Armstrong Number

    An Armstrong number is a number that is equal to the sum of its own digits raised to the power of the number of digits.

    #include <stdio.h>
#include <math.h>

int main() {
    int num, sum = 0, temp, remainder, n = 0;
    printf("Enter a number: ");
    scanf("%d", &num);
    temp = num;
    while (temp != 0) {
        temp /= 10;
        ++n;
    }
    temp = num;
    while (temp != 0) {
        remainder = temp % 10;
        sum += pow(remainder, n);
        temp /= 10;
    }
    if (sum == num) {
        printf("%d is an Armstrong number\n", num);
    } else {
        printf("%d is not an Armstrong number\n", num);
    }
    return 0;
}

    19. Program to Merge Two Arrays

    This program merges two arrays into a third array.

    #include <stdio.h>

int main() {
    int n1, n2;
    printf("Enter number of elements in first array: ");
    scanf("%d", &n1);
    int arr1[n1];
    printf("Enter elements of first array: ");
    for (int i = 0; i < n1; i++) {
        scanf("%d", &arr1[i]);
    }

    printf("Enter number of elements in second array: ");
    scanf("%d", &n2);
    int arr2[n2];
    printf("Enter elements of second array: ");
    for (int i = 0; i < n2; i++) {
        scanf("%d", &arr2[i]);
    }

    int merged[n1 + n2];
    for (int i = 0; i < n1; i++) {
        merged[i] = arr1[i];
    }
    for (int i = 0; i < n2; i++) {
        merged[n1 + i] = arr2[i];
    }

    printf("Merged Array: ");
    for (int i = 0; i < n1 + n2; i++) {
        printf("%d ", merged[i]);
    }
    printf("\n");
    return 0;
}

    20. Program to Sort an Array Using Bubble Sort

    This program sorts an array using the Bubble Sort algorithm.

    #include <stdio.h>

int main() {
    int n, temp;
    printf("Enter the number of elements: ");
    scanf("%d", &n);
    int arr[n];
    printf("Enter the elements: ");
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }

    printf("Sorted Array: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}
