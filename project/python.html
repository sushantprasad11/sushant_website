Python Programming Notes

1. Introduction to Python

Python is an interpreted, high-level programming language known for its simplicity and readability. It was created by Guido van Rossum and released in 1991. Python's syntax allows developers to write clear, logical code for small and large-scale projects.

2. Python Basics

Python uses dynamic typing, meaning you don’t need to declare variable types. Python is also known for its rich standard library that supports many functionalities like file I/O, system calls, and internet protocols.

Example Code:

# Python program to print a message
print("Hello, World!")

3. Variables and Data Types

In Python, variables are dynamically typed, meaning you don’t need to declare their type. Python has several built-in data types, such as integers, floats, strings, lists, tuples, and dictionaries.

Example Code:

# Defining variables of different types
a = 10           # integer
b = 3.14         # float
c = "Python"     # string
d = [1, 2, 3]    # list

# Printing variable types
print(type(a))
print(type(b))
print(type(c))
print(type(d))

4. Input and Output

Python's input() function allows user input, and the print() function is used to output data to the console.

Example Code:

# Input and Output Example
name = input("Enter your name: ")
print("Hello, " + name)

5. Operators in Python

Python supports several operators such as arithmetic operators (+, -, *, /), comparison operators (==, !=, >, <), logical operators (and, or, not), and others.

Example Code:

# Arithmetic operators
x = 5
y = 3
sum = x + y
product = x * y

# Comparison operators
result = (x > y)

print("Sum: ", sum)
print("Product: ", product)
print("Is x greater than y? ", result)

6. Conditional Statements

Python supports if, else, and elif to make decisions based on conditions.

Example Code:

# Conditional statement example
num = int(input("Enter a number: "))
if num > 0:
    print("Positive number")
elif num < 0:
    print("Negative number")
else:
    print("Zero")

7. Loops in Python

Python supports two primary types of loops: for and while.

Example Code:

# For loop example
for i in range(5):
    print(i)

# While loop example
n = 5
while n > 0:
    print(n)
    n -= 1

8. Functions in Python

Functions in Python are defined using the def keyword. Functions can return values or perform actions.

Example Code:

# Function to add two numbers
def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print("Sum:", result)

9. Lists in Python

Lists in Python are mutable, ordered collections of elements. They can contain elements of different types and are defined using square brackets [].

Example Code:

# Creating and manipulating lists
my_list = [1, 2, 3, 4]
my_list.append(5)
my_list.remove(2)
print("Updated List:", my_list)

10. Dictionaries in Python

Dictionaries are unordered collections of key-value pairs. They are defined using curly braces {}.

Example Code:

# Creating and accessing a dictionary
my_dict = {"name": "John", "age": 25, "city": "New York"}
print("Name:", my_dict["name"])
print("Age:", my_dict["age"])

11. Tuples in Python

Tuples are immutable sequences in Python. Once defined, their elements cannot be changed. Tuples are defined using parentheses ().

Example Code:

# Creating and accessing a tuple
my_tuple = (1, 2, 3)
print("First Element:", my_tuple[0])

12. Sets in Python

Sets are unordered collections of unique elements. They are defined using curly braces {}.

Example Code:

# Creating and manipulating a set
my_set = {1, 2, 3, 4}
my_set.add(5)
my_set.remove(2)
print("Updated Set:", my_set)

13. File Handling in Python

Python provides functions for file handling, such as opening, reading, writing, and closing files.

Example Code:

# File handling example
file = open("sample.txt", "w")
file.write("Hello, Python!")
file.close()

file = open("sample.txt", "r")
print(file.read())
file.close()

14. Exception Handling in Python

Python provides try, except, and finally blocks to handle exceptions and errors.

Example Code:

# Exception handling example
try:
    num = int(input("Enter a number: "))
    print("You entered:", num)
except ValueError:
    print("Invalid input! Please enter an integer.")
finally:
    print("Execution completed.")

15. Classes and Objects in Python

Python supports Object-Oriented Programming (OOP) features like classes, objects, inheritance, etc. Classes are defined using the class keyword.

Example Code:

# Class and Object example
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is", self.name)

person1 = Person("Alice", 30)
person1.greet()

16. Modules and Packages in Python

Modules allow you to logically organize your Python code, and packages help organize multiple modules into directories.

Example Code:

# Importing the math module
import math

result = math.sqrt(16)
print("Square root of 16 is:", result)

Conclusion

Python is a powerful, versatile language that is easy to learn and widely used in many fields such as web development, data science, and automation. This brief guide covers the basics, but Python has many advanced features and libraries that make it an excellent choice for developers.

25. Decorators

def decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()
